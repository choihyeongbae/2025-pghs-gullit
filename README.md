# 어린이 교통사고 안전도 분석 시스템 🚦

> **제5회 학생 SW-AI 인재 양성 프로젝트**  
> 포항여자고등학교 굴리트팀

전국 225개 시군구의 어린이 교통사고 데이터를 K-means 클러스터링으로 분석하여 지역별 안전도를 시각화한 인터랙티브 대시보드입니다.

---

## 📊 주요 기능

### 1. 지능형 군집 분석
- **K-means 클러스터링** (K=3)으로 지역을 3개 그룹으로 자동 분류
- 10회 반복 실행 후 최적 결과 선택 (Inertia 기준)
- **실루엣 스코어**로 클러스터링 품질 평가
- 표준화된 데이터로 정확한 거리 계산

### 2. 인터랙티브 대시보드
- **실시간 검색**: 지역명 입력 시 즉시 필터링
- **상세보기 모드**: 특정 지역 선택 시 심층 분석
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화
- **스무스 스크롤**: 섹션 간 부드러운 이동

### 3. 다층 시각화
```
📈 Scatter Plot
   ├─ 일반 클러스터 점 (레이어 3)
   ├─ 중심점 표시 (레이어 2)
   └─ 선택 지역 강조 (레이어 1) ⭐

📊 Bar Chart
   └─ 클러스터별 평균 사고율

📋 Ranking Tables
   ├─ 부상자수 상위/하위 10개
   └─ 사고건수 상위/하위 10개
```

### 4. 군집별 분류

| 군집 | 레벨 | 특징 | 색상 |
|-----|------|-----|------|
| **0** | 🟢 안전 | 낮은 사고율 지역 | 초록색 |
| **1** | 🟡 주의 | 중간 사고율 지역 | 주황색 |
| **2** | 🔴 위험 | 높은 사고율 지역 | 빨간색 |

## 💡 핵심 알고리즘

### K-means 클러스터링 구현
```javascript
// 1. 데이터 표준화 (Z-score normalization)
normalized = (value - mean) / std

// 2. K-means++ 초기화
// - 첫 번째 중심점: 랜덤 선택
// - 나머지: 거리 기반 확률적 선택

// 3. 반복 수렴
while (!converged && iteration < maxIterations) {
    // 각 점을 가장 가까운 중심점에 할당
    // 중심점 업데이트
}

// 4. 품질 평가
- Inertia: 클러스터 내 거리 합
- Silhouette Score: 클러스터 분리도
```

### 실루엣 스코어 계산
```
s(i) = (b(i) - a(i)) / max(a(i), b(i))

where:
  a(i) = 같은 클러스터 내 평균 거리
  b(i) = 가장 가까운 다른 클러스터까지의 평균 거리
```

## 🔬 분석 결과

### 클러스터 특성

| 군집 | 평균 부상자수 | 평균 사고건수 | 지역 수 | 비율 |
|-----|-------------|-------------|--------|------|
| 0 (안전) | ~1.5명 | ~1.2건 | ~110개 | ~49% |
| 1 (주의) | ~3.0명 | ~2.5건 | ~85개 | ~38% |
| 2 (위험) | ~6.0명 | ~4.5건 | ~30개 | ~13% |

### 주요 발견 사항

1. **지리적 특성**
   - 대도시권: 주로 안전/주의 군집
   - 농어촌 지역: 일부 위험 군집 분포

2. **사고 패턴**
   - 부상자수와 사고건수의 높은 상관관계
   - 일부 지역: 사고건수 대비 높은 부상자수

3. **정책 시사점**
   - 위험 군집 지역: 즉각적 안전 대책 필요
   - 주의 군집 지역: 예방적 조치 권장
